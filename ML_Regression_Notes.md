# Machine Learning Regression Notes

## Overview
Regression analysis is used for predicting a continuous outcome variable based on one or more predictor variables.

## Types of Regression
1. **Linear Regression**: Models the relationship between two variables by fitting a linear equation.
2. **Multiple Regression**: Extension of linear regression that uses multiple predictors.
3. **Polynomial Regression**: Models the relationship as an nth degree polynomial.
4. **Ridge Regression**: A type of linear regression that includes a penalty to the size of coefficients.
5. **Lasso Regression**: Similar to ridge, but can also reduce some coefficients to zero, effectively selecting variables.

## Key Concepts
- **Coefficients**: Indicate the strength and type of relationship between each predictor and the outcome.
- **R-squared**: A statistical measure of how close the data are to the fitted regression line.
- **Residuals**: The differences between the observed and predicted values.
