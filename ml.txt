# 📘 Machine Learning – Regression (Interview Notes)

---

# 🔴 1. Linear Regression Model

## Model Formulation

y = Xβ + ε

* y: target
* X: feature matrix
* β: coefficients
* ε: error

---

## Closed Form Solution (OLS)

β = (XᵀX)⁻¹ Xᵀy

### Key Points

* Derived by minimizing squared error
* Gives Best Linear Unbiased Estimator (BLUE)

### Limitations

* Fails if XᵀX is non-invertible (multicollinearity)
* Expensive for large datasets

---

# 🔴 2. Loss Functions

## 1. Mean Squared Error (MSE)

MSE = (1/n) Σ(y - ŷ)²

### Use

* When large errors are costly
* Smooth & convex

### Problem

* Sensitive to outliers

---

## 2. Mean Absolute Error (MAE)

MAE = (1/n) Σ|y - ŷ|

### Use

* When outliers exist

### Problem

* Not differentiable at zero

---

## 3. RMSE

RMSE = √MSE

### Use

* Same unit as target
* Better interpretability

---

## 4. Huber Loss

* MSE for small errors
* MAE for large errors

### Use

* Real-world noisy datasets

---

## 5. Quantile Loss

Used to predict percentiles

### Use

* Forecasting (important)

---

# 🔴 3. Optimization

## Gradient Descent

θ = θ - α ∇J(θ)

### Types

* Batch
* Stochastic
* Mini-batch (best in practice)

---

# 🔴 4. Assumptions of Linear Regression

## 1. Linearity

* Relationship must be linear
* Violation → underfitting

Fix:

* Polynomial features
* Tree models

---

## 2. Independence of Errors

* No autocorrelation

Violation:

* Common in time series

Fix:

* Lag features
* Time-series models

---

## 3. Homoscedasticity

* Constant variance of residuals

Violation:

* Funnel shape

Fix:

* Log transformation
* Weighted regression

---

## 4. No Multicollinearity

* Features should not be correlated

Violation:

* Unstable coefficients

Fix:

* Remove features
* Ridge regression
* PCA

---

## 5. Normality of Residuals

* Needed for statistical inference

Violation:

* Affects confidence intervals

Fix:

* Transform target

---

# 🔴 5. Regularization

## Ridge (L2)

Loss = MSE + λ Σβ²

* Shrinks coefficients
* Handles multicollinearity

---

## Lasso (L1)

Loss = MSE + λ Σ|β|

* Feature selection
* Produces sparse model

---

## ElasticNet

Combination of Ridge + Lasso

---

# 🔴 6. Bias-Variance Tradeoff

* Bias: error due to simplicity
* Variance: error due to complexity

Goal:

* Balance both

---

# 🔴 7. Diagnostics

## Residual Plots

* Detect non-linearity
* Detect heteroscedasticity

## Q-Q Plot

* Check normality

## VIF

VIF = 1 / (1 - R²)

* > 10 indicates multicollinearity

---

# 🔴 8. Important Edge Cases

* XᵀX not invertible → use Ridge
* Large dataset → use SGD
* Outliers → use MAE/Huber
* Skewed data → log transform

---

## Intermediate

---

## Advanced

### Why MSE penalizes outliers?

Because errors are squared

---

### Ridge vs Lasso

* Ridge shrinks
* Lasso selects features

---

### Why Lasso gives sparse solution?

L1 penalty forces coefficients to zero

---

### What is bias-variance tradeoff?

Balance between underfitting and overfitting

---

### Why R² is not reliable?

Always increases with more features

---

### Adjusted R²

Penalizes unnecessary features

---

### When to use MAE vs RMSE?

* MAE → outliers
* RMSE → large errors matter

---

### Why Linear Regression is sensitive to outliers?

Because of squared error

---

### What if assumptions fail?

* Model inference becomes unreliable
* Predictions may degrade
